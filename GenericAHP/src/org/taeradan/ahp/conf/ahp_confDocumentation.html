Element Details

root

The root element of the XML document is the root element of the AHP tree. It must contain a name, a preference matrix and an array of criterias elements.

prefmatrix

The preference matrix is a square matrix that establish preferences between each criterias in the root or each indicators of a given criteria. Contains rows. It must have as much rows and colums than the number of elements compared.
